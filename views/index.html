Omnispace Hamburger Menu Animation Challenge

This is what came of a design challenge to create a few sleek hamburger icon animations.

How to use

  1. Download a zip of the GitHub repository
  2. Import hamburger-styles.css and hamburger-icon.js into the desired page
  3. Copy and paste html from index.html of the type of icon you want to use
